# Bang Bang Auto Detailing

## Project Overview
Business website for Bang Bang Auto Detailing — a mobile auto detailing service.

## Tech Stack
- **Static Site Generator**: Eleventy (11ty) v3
- **Templates**: Nunjucks (`.njk`)
- **Styling**: CSS
- **Hosting**: Static

## Development Commands
```bash
npm start        # Dev server with live reload
npm run build    # Production build
```

## File Structure
```
pages/           # Page content
_layouts/        # Page layouts
_includes/       # Reusable partials/components
assets/          # Static assets (images, CSS, JS)
.eleventy.js     # Eleventy configuration
.kiro/           # Kiro workflow specs and templates
```
